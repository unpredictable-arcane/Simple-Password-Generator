# Password Generator

A simple and modern password generator built with **HTML**, **CSS**, and **JavaScript**.  
Features a responsive glass-style container, animated background, adjustable options, and crypto-secure randomness.

---

## Features
- Animated gradient background  
- Glassmorphism-style container box  
- Adjustable password length (6â€“64 characters)  
- Toggle lowercase / uppercase / numbers / symbols  
- Copy to clipboard button  
- Password strength indicator  
- Uses `crypto.getRandomValues()` for randomness  
- Fully client-side (no data sent anywhere)
